# 🔍 Monitoramento Automatizado de Sites

Projeto desenvolvido para monitorar disponibilidade de sistemas web e validar acessos automaticamente.

---

## 🚀 Funcionalidades

* ✔ Verificação de disponibilidade (online/offline)
* ✔ Automação de login em sistemas web
* ✔ Timeout configurável
* ✔ Estrutura baseada em JSON (flexível)
* ✔ Geração de logs

---

## 🛠️ Tecnologias utilizadas

* Python
* Playwright
* JSON
* Automação Web

---

## 📌 Exemplo de configuração

```json
{
  "nome": "Google",
  "url": "https://www.google.com",
  "apenas_verificar_online": true
}
```

---

## ▶️ Como executar

1. Instale o Python
2. Instale as dependências:

```
pip install playwright
playwright install
```

3. Execute o projeto:

```
python monitorar.py
```

---

## 📊 Aplicações

* Monitoramento de sistemas corporativos
* Validação de disponibilidade de serviços
* Automação de testes simples
* Rotinas de verificação em TI
